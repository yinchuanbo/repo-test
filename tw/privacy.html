    隱私政策

    本隱私政策適用於 Vidqu Inc. 擁有和運營的網站（“Vidqu.com”）、其子網站、軟體或應用程式及任何附加服務（“網站”、“服務”、“產品”、“Vidqu”、“我們”或“我們的”）。

    本隱私政策的目的是告知您在訪問我們的網站和/或使用我們的產品和服務時，我們收集、使用和分享哪些信息。您的隱私對我們來說非常重要，我們承諾始終保護您的隱私。

    在訪問我們的網站、向我們提交您的數據以及訪問、安裝和/或使用我們的產品和服務時，請仔細閱讀本隱私政策並確保您了解其中內容。

    您第一次使用我們的網站，即表示您接受我們的隱私政策。如果您不同意本隱私政策，您必須立即停止使用我們的網站。

    我們收集什麼信息

    以下列出的所有信息都是用於提供和改進服務。我們不會在本隱私政策描述之外使用或分享您的信息。為最大限度地保護您的隱私，Vidqu 僅收集提供更好用戶體驗所需的最少信息。

    服務數據

    為了向您提供我們的產品和服務，我們會收集客戶的有限個人信息，包括但不限於以下信息：

    1. 1. 電子郵件地址（僅在同意的情況下）
    2. 2. 用戶名（僅在同意的情況下）
    3. 3. IP 地址（自動收集）
    4. 4. 網頁瀏覽器類型和版本（自動收集）
    5. 5. 操作系統（自動收集）
    6. 6. 以參考網站開頭的 URL 列表，您在我們網站上的活動
    客戶服務和技術服務

    當您通過電子郵件或請求在線支持聯繫我們時，我們可能會收集或要求您提供必要的信息。

    這些信息可能包括但不限於您的姓名、您的電子郵件地址、您的訂單參考、您如何使用我們的服務、您在使用我們的服務時與我們的服務的互動方式、您在我們平台上的操作，如您錄製的視頻數量、您的分享活動或您啟用的第三方集成、您的計算機或設備信息如硬件和系統版本、崩潰日誌、截圖、視頻錄像等。一般來說，我們收集的信息不會超出調查您遇到的問題所需的信息。我們僅會使用此類信息來提供客戶服務和/或解決您遇到的技術問題。

    Cookies

    當您訪問我們的網站時，我們可能會在您的計算機上存儲 cookie（以及其他跟踪技術，如網絡信標和像素），以便促進和定制您對我們網站的使用。cookie 是一個小的數據文本文件，當您的網頁瀏覽器允許時，網站會將其存儲在您的計算機硬盤上，稍後可以檢索以識別您。我们的cookie可能会存储有关您账户的信息。如果您的瀏覽器允許，您可以隨時拒絕我們的 cookie，但這樣做可能會導致我們網站的一些功能無法正常運行，並且您的在線體驗可能會受到限制。您可以選擇隨時刪除 cookie；然而，您可能會失去任何使您能夠更快速高效地訪問我們網站的信息，包括但不限於登錄和個性化設置。我們建議您保持您的網絡瀏覽器和操作系統的最新狀態，如果您不確定如何調整隱私設置，請咨詢您的網絡瀏覽器開發人員和計算機或設備製造商提供的幫助和指導。

    我們從第三方收到的信息：

    Vidqu 可能會與第三方合作，以為您提供無縫的使用體驗。第三方可能會與我們分享您的某些信息，例如您的姓名、電子郵件地址或其他使集成成為可能的必要信息。此外，如果您使用我們的單一登錄提供商（如 Google、LinkedIn 和 Facebook）註冊或登錄我們的服務，我們會收集提供商向我們提供的身份驗證數據，以便您登錄。

    下訂單

    當您購買我們的產品或服務時，帳單處理由代表我們行事的第三方服務提供商處理；因此，您將直接向服務提供商購買，而不是向 Vidqu 購買。您個人信息的處理將受服務提供商發布的任何隱私政策或服務條款的約束。

    服務提供商可能會要求帳單信息，例如您的姓名、電子郵件地址、家庭或工作地址、電話號碼或手機號碼、付款信息，包括信用卡號碼、到期日期、安全碼、發卡銀行名稱等。此信息的目的是履行訂單、處理付款、交付產品、提供客戶服務等。第三方服務提供商可能會向我們傳送您的部分個人信息，但不包括付款信息。我們可能會使用我們收到的信息來驗證您的訂閱情況，聯繫您了解訂單狀態，以及在適用情況下續訂您的訂閱。

    我們與所有第三方服務提供商簽訂了協議，他們不會將我們用戶的個人信息用於他們自己的營銷目的，並且不會與其他方分享這些信息。

    使用社交網絡

    如果您選擇通過 Facebook 或其他社交網絡“分享” Vidqu 產品，您可能會收到來自這些網站的 cookie。我們無法控制這些 cookie 的設置，因此請查看這些網站以了解有關其 cookie 及其管理方式的更多信息。

    附加說明：一旦 Vidqu 視頻發布到其他第三方平台，內容將受這些平台的約束和管理。

    我們如何使用您的信息

    我們使用您的信息是為了向您提供服務。我們從您那裡收集的任何信息可能會用於以下其中一種方式：

    提供請求的服務

    某些數據是提供您請求的服務所必需的：

    1. 1. 提供和管理您的帳戶
    2. 2. 提供和管理您對我們網站的訪問
    3. 3. 個性化和定制您在我們網站上的體驗
    4. 4. 向您提供我們的產品和服務
    5. 5. 個性化和定制我們的產品和服務
    6. 6. 回應您的通信

    向您提供您訂閱的電子郵件，例如新聞簡報、提醒等（您可以隨時通過點擊我們電子郵件底部的“取消訂閱”來取消訂閱或退出）。

    改進我們的網站和服務

    通過分析您對我們網站的使用並收集反饋，我們不斷努力根據我們從您那裡收到的信息和反饋改進我們的網站產品。所有信息都是完全匿名的，您的真實信息不會被曝光或使用。

    改進客戶服務

    您的信息幫助我們更有效地回應您的客戶服務請求和支持需求。

    處理交易

    您的信息，不論是公開還是私人，都不會在未經您同意的情況下出售、交換、轉讓或提供給任何其他公司，僅用於交付所購買的產品或所請求的服務。 在得到您的許可和/或法律允許的情況下，我們可能還會將您的數據用於營銷目的，包括通過電子郵件聯繫您，向您提供有關我們產品和服務的新聞和優惠。

    發送定期電子郵件

    您提供的訂單處理電子郵件地址將僅用於向您發送與您訂單相關的信息和更新。如果您希望隨時取消訂閱未來的電子郵件，您可以按照我們在每封電子郵件底部包含的詳細取消訂閱說明進行操作。

    我們如何以及在哪裡存儲您的數據？

    我們僅在需要根據規則使用您的數據和/或我們獲得您允許保留它的情況下保留您的數據。無論如何，我們將在適當的時候進行審查，以確定是否需要保留您的數據。如果根據我們的數據保留政策不再需要您的數據，您的數據將被刪除。

    我們的數據保留政策如下：

    1. 1. 您的數據將在提出請求後 72 小時內刪除。
    2. 2. 訂閱者數據將在點擊我們公司發送的任何電子郵件底部的 "取消訂閱" 後或手動請求後立即刪除。
    3. 3. 我們非常重視數據安全，為保護您的數據，我們已經制定了適當的物理、電子和管理程序來保護和保障通過我們網站收集的數據。
    4. 4. 儘管我們採取了安全措施，但重要的是要記住，通過互聯網傳輸數據可能不是完全安全的，建議您在通過互聯網向我們傳輸數據時採取適當的預防措施。

    我們如何分享您的信息

    Vidqu 不會將您的個人信息與任何第三方分享或披露，但有以下兩種例外情況：

    第三方服務提供商

    我們可能會聘請其他公司代表我們提供有限的服務，例如處理和交付郵件、提供客戶支持、託管網站、處理交易或對我們的服務進行統計分析。這些公司將只允許獲得他們需要的個人信息以提供服務。他們被要求保密這些信息，不得將其用於任何其他目的。

    法律要求

    如果我們收到有效的傳票、搜查令或其他法律文件（例如，有關我們全部或部分業務出售的文件），或適用法律要求我們遵守，或當我們需要保護 Vidqu Inc. 的權利或財產，包括我們產品和服務的安全性以及我們客戶和員工的人身安全、財產或其他權利，或為了識別和/或防止信用卡欺詐而提出的數據請求時，我們可能會分享您的個人數據。

    我們網站的使用

    我們可能會編制有關我們網站使用情況的統計數據，包括流量數據、使用模式、用戶數量、銷售數據和其他信息。所有這些數據將是匿名的，且不會包含任何個人身份信息。

    安全性和機密性

    我們理解並接受保護您數據的責任。我們的員工承擔保密義務。我們使用行業標準的信息安全工具和措施，以及內部程序和嚴格的指導方針來防止信息濫用和數據洩漏，但我們不能保證我們的系統將是絕對安全的（實際上，沒有任何一種通過互聯網傳輸或電子存儲的方法是 100% 安全的）。如果您發現任何潛在的數據洩露或安全漏洞，請立即與我們聯繫。我們將採取所有措施來調查事件，包括必要的預防措施。

    此外，極為重要的是，您需要了解保護您註冊 Vidqu 服務帳戶的密碼安全是您的責任。密碼應該強且唯一，不易被猜到。密碼應該只有您知道。您還應該在安全的地方保留一份副本以供參考，因為您需要密碼來解密保存在我們服務中的所有數據。我們無法解密您的數據。

    如何訪問、控制和刪除您的數據

    當您通過我們的網站提交信息時，您可能會獲得限制我們使用您的數據的選項。我們旨在讓您對我們使用您的數據擁有強大的控制權（包括您可以通過使用我們電子郵件中提供的鏈接來選擇退出接收我們的電子郵件）。

    您可以隨時通過登錄您的帳戶，添加、刪除、編輯和更改保存在 Vidqu 服務中的任何數據。

    您還有權隨時通過聯繫我們 support@vidqu.ai 停用和刪除您的 Vidqu 帳戶。然而，一些信息可能會保留在存檔/備份副本中，以符合我們的法律義務、解決爭議和執行我們的協議。

    對於由第三方合作夥伴處理的信息，您有權訪問您的個人信息。如果您想訪問、更正、修改或刪除第三方合作夥伴平台上的不準確數據，您應直接聯繫他們，因為他們是數據控制者，Vidqu 無法控制這些信息。

    信息保證免責聲明

    儘管本政策中有任何其他規定，我們對我們收集、存儲和披露給您或其他人的任何信息的準確性、正確性和安全性不承擔任何責任。

    我們的政策僅針對我們從您那裡收集的信息的使用和披露。只要您在互聯網上向其他方或網站披露您的信息，不同的規則可能適用於他們對您向他們披露的信息的使用或披露。因此，我們建議您閱讀每個您選擇向其披露信息的第三方的條款和條件及隱私政策。

    兒童在線隱私保護法合規性

    我們遵守 COPPA（兒童在線隱私保護法）的要求，我們不會收集任何 13 歲以下人員的信息。我們的網站、產品和服務均針對至少 13 歲或以上的人士。如果您是家長或監護人並且知道您的孩子向我們提供了個人信息，請聯繫我們。如果我們得知我們在未經父母同意的情況下從 18 歲以下的兒童那裡收集了個人信息，我們將採取措施從我們的服務器中刪除這些信息。

    隱私政策的變更

    我們可能會不時更新我們的隱私政策。因此，建議您定期查看此頁面以了解任何變更。這些變更在發布到此頁面後立即生效。最新修訂將反映在“最後更新”標題中。此政策的任何變更將發布在網站上。您在此類變更後繼續使用服務將被視為您同意修訂後的隱私政策。

    如何聯繫我們

    如果您對本隱私政策有任何疑問或認為 Vidqu 沒有遵守本隱私政策，請發電子郵件至 support@vidqu.ai，我們將努力及時確定並解決問題。
