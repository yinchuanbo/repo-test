    収集する情報

    当社が収集する以下のすべての情報は、サービスの提供と改善のために使用されます。当社は、このプライバシーポリシーに記載されている場合を除き、お客様の情報を使用したり、第三者と共有したりすることはありません。お客様のプライバシーを最大限に高めるため、Vidqu はより良いユーザーエクスペリエンスを提供するために必要な最小限の情報のみを収集します。

    サービスデータ

    お客様に当社の製品とサービスを提供するために、当社はお客様から以下の情報を含むがこれらに限定されない限られた個人情報を収集します:

    1. 1. メールアドレス (同意を得た場合のみ)
    2. 2. ユーザー名 (同意を得た場合のみ)
    3. 3. IP アドレス (自動的に収集)
    4. 4. Web ブラウザの種類とバージョン (自動的に収集)
    5. 5. オペレーティングシステム（自動的に収集されます）
    6. 6. 参照サイトから始まる URL のリスト、当サイトでのお客様のアクティビティ
    カスタマーサービスとテクニカルサービス

    お客様が電子メールで当社に連絡するか、オンラインサポートをリクエストする際に、必要な情報を収集するか、お客様に提供をお願いする場合があります。

    この情報には、お客様の名前、電子メールアドレス、注文番号、当社のサービスの使用方法、当社のサービスを使用する際の当社のサービスとのやり取り、当社のプラットフォームで行ったアクション（録画したビデオの数、共有アクティビティ、有効にしたサードパーティ統合など）、ハードウェアやシステムのバージョン、クラッシュログ、スクリーンショット、ビデオ録画などのコンピューターまたはデバイスの情報が含まれますが、これらに限定されません。通常、お客様が遭遇した問題を調査するために必要な情報以上の情報は収集しません。当社は、このような情報を、カスタマーサービスの提供および/またはお客様が経験している技術的な問題のトラブルシューティングにのみ使用します。

    Cookies

    お客様が当社の Web サイトにアクセスすると、お客様のコンピューターに Cookie が保存されることがあります（Web トラッキングなどの他の追跡技術も同様です）。弊社は、弊社のウェブサイトの利用を容易にし、カスタマイズするために、クッキー（ビーコンやピクセルなど）を使用しています。クッキーとは、ウェブサイトがお客様のコンピューターのハードドライブに保存する小さなデータテキストファイルで（お客様のウェブブラウザが許可している場合）、後で取得してお客様を弊社が識別することができます。弊社のクッキーには、お客様のアカウント情報に関する情報が保存される場合があります。ブラウザが許可している場合は、いつでもクッキーを拒否できますが、その場合、弊社のウェブサイトの一部の機能が正しく動作せず、オンラインエクスペリエンスが制限される可能性があります。いつでもクッキーを削除することができますが、ログインやパーソナライズ設定など、より迅速かつ効率的に弊社のサイトにアクセスできるようにする情報が失われる可能性があります。インターネットブラウザとオペレーティングシステムを最新の状態に保ち、プライバシー設定の調整について不明な点がある場合は、インターネットブラウザの開発者やコンピューターまたはデバイスの製造元が提供するヘルプとガイダンスを参照することをお勧めします。

    第三者から受け取る情報：

    Vidqu は、シームレスなエクスペリエンスを提供するために第三者と相互運用する場合があります。第三者は、お客様の名前、メールアドレス、または統合を可能にするために必要なその他の情報など、お客様の情報の一部を当社と共有する場合があります。さらに、お客様が当社のシングルサインオンプロバイダー（Google、LinkedIn、Facebookなど）のいずれかを使用して当社のサービスに登録またはログインする場合、当社はお客様がログインできるようにプロバイダーが提供する認証データを収集します。

    注文

    お客様が当社の製品またはサービスを購入すると、請求は当社に代わって行動するサードパーティのサービスプロバイダーによって処理されます。したがって、お客様はVidquからではなく、サービスプロバイダーから直接購入することになります。お客様の個人情報の取り扱いは、サービスプロバイダーが公開するプライバシーポリシーまたは利用規約に準拠します。

    サービスプロバイダーは、お客様の名前、メールアドレス、自宅または勤務先住所、電話番号または携帯電話番号、クレジットカード番号、有効期限、セキュリティコード、発行銀行名などの支払い情報などの請求情報を要求する場合があります。この情報の目的は、注文の履行、支払いの処理、製品の配送、顧客サービスの提供などです。サードパーティのサービスプロバイダーは、お客様の個人情報の一部を当社に送信することがありますが、支払い情報は含まれません。当社は、お客様のサブスクリプションの確認、注文状況に関するご連絡、および該当する場合はサブスクリプションの更新のために、受け取った情報を使用する場合があります。

    当社は、すべてのサードパーティのサービスプロバイダーと、当社のユーザーの個人情報を自社のマーケティング目的で使用せず、この情報を他の当事者と共有しないという契約を結んでいます。

    ソーシャルネットワークの使用

    Facebook またはその他のソーシャルネットワークを通じて Vidqu 製品を友人と「共有」することを選択した場合、これらの Web サイトから Cookie が送信される可能性があります。当社はこれらの Cookie の設定を管理していませんので、Cookie の詳細と管理方法については、それらの Web サイトを確認してください。

    追加メモ: Vidqu のビデオが他のサードパーティプラットフォームに公開されると、コンテンツはプラットフォームの制約と管理の対象となります。

    お客様の情報の使用方法

    当社は、お客様にサービスを提供するためにお客様の情報を使用します。当社がお客様から収集した情報は、次のいずれかの方法で使用される場合があります:

    リクエストされたサービスを提供するため

    お客様が要求したサービスを提供するために、いくつかのデータが必要です:

    1. 1. お客様のアカウントの提供と管理
    2. 2. 当サイトへのアクセスの提供と管理
    3. 3. 当サイトでのお客様の体験をパーソナライズし、カスタマイズ
    4. 4. お客様への製品とサービスの提供
    5. 5. お客様への製品とサービスのパーソナライズとカスタマイズ
    6. 6. お客様からの連絡への対応

    お客様が購読しているメール（ニュースレター、アラートなど）をお客様に提供します（お客様は、当社のメールの下部にある「購読解除」をクリックすることで、いつでも購読を解除またはオプトアウトできます）。

    当社のウェブサイトとサービスを改善するため

    当社は、お客様の当サイトの利用状況を分析し、フィードバックを収集することで、お客様から受け取った情報とフィードバックに基づいて、当社のウェブサイトの提供内容を継続的に改善するよう努めています。すべての情報完全に匿名化されており、実際の情報が公開または使用されることはありません。

    カスタマーサービスを改善するため

    お客様の情報は、お客様のカスタマー サービス リクエストやサポート ニーズに、より効果的に対応するために役立ちます。

    取引を処理するため

    お客様の情報は、公開されているか非公開であるかにかかわらず、購入された製品または要求されたサービスを提供するという明確な目的以外で、お客様の同意なしに、いかなる理由でも他の会社に販売、交換、譲渡、または提供されることはありません。お客様の許可がある場合、および/または法律で許可されている場合、当社は、製品やサービスに関するニュースやオファーを電子メールでお客様に連絡することを含め、マーケティング目的でお客様のデータを使用する場合もあります。

    定期的なメールを送信するため

    注文処理のために提供された電子メール アドレスは、注文に関する情報や更新情報をお客様に送信するためにのみ使用されます。今後の電子メールの受信を停止したい場合は、各電子メールの下部に記載されている詳細な登録解除手順に従ってください。

    お客様のデータをどのように、どこに保存しますか？

    当社は、規則に従って使用する必要がある期間、および/またはお客様の許可がある期間のみ、お客様のデータを保管します。いずれにせよ、当社は、お客様のデータを保管する必要があるかどうかを確認するために、適切な場合にレビューを実施します。当社のデータ保管ポリシーの条件により、お客様のデータが不要になった場合は、お客様のデータは削除されます。

    当社のデータ保管ポリシーは次のとおりです:

    1. 1. リクエストに応じて、データは 72 時間以内に削除されます。
    2. 2.購読者のデータは、当社から送信された電子メールの下部にある「購読解除」をクリックするか、手動でリクエストすると、直ちに削除されます。
    3. 3. データのセキュリティは当社にとって非常に重要であり、お客様のデータを保護するために、当社では、サイトを通じて収集されたデータを保護およびセキュリティで保護するための適切な物理的、電子的、および管理上の手順を実施しています。
    4. 4. 当社が講じているセキュリティ対策にもかかわらず、インターネット経由のデータ送信は完全に安全ではない可能性があることに留意することが重要です。インターネット経由で当社にデータを送信する場合は、適切な予防策を講じることをお勧めします。
